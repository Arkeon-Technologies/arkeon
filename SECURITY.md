# Security Policy

## Reporting a Vulnerability

If you believe you've found a security vulnerability in Arkeon, please report it privately rather than opening a public issue.

Email: admin@arkeon.tech

Please include:
- A description of the vulnerability
- Steps to reproduce
- The affected version(s)
- Any potential impact you've identified

We'll acknowledge receipt within 72 hours and aim to provide an initial assessment within one week.

## Supported Versions

During pre-1.0 development, only the latest minor version receives security updates.
